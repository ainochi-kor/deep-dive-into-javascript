# Webpack 공부 중

> $ npx webpack --mode production --entry ./source/index.js -o ./public --output-filename index_bundle.js

- `--mode`: 환경에 대한 설정 [`production`, `development`, `none`]
- `--entry`: 번들링 하고 싶은 파일
- `-o`: 번들링 완료한 파일의 생성 위치
- `--output-filename`: 번들링 완료한 파일의 이름
